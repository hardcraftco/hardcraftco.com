== Description ==

If you are looking for a solution which will help you add social media icons to your page then you sholud test this one. Musli Social Links is user friendly WordPress plugin created to help you add social media icons to your webpage. It have no limits in number of links (icons) you can add and manage. It's easy to use and very intuitive, but also offers pretty nice set of themes and customization options.

Its based on FontAwesome, therefore it comes with many, many social media icons (over 40 most popular icons are integrated with plugin). 7 different themes.

== Installation ==

1. Upload `musli-social-links` directory to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Add and configure tabs (Musli Links -> Manage Links), configure plugin (Musli Links -> General Settings).

== Frequently Asked Questions ==

= Saving options not change anything on the frontend page =

This problem might occur if your browser cached page. Try to clean up cache from your browser, or try another browser.

= Cleaning browser's cache didn't helped, what's next? =

Check if you have some caching plugins installed. Try to clean up your site's cache.
