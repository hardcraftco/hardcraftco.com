ABOUT Theme
=========== 

Shopaholic is a clean and subtle eCommerce theme which puts the focus firmly on your products. You'll want to make sure your product imagery is tip top. Shopaholic is a child theme of the excellent Storefront theme from the creators of WooCommerce and it supports all the same features and functions of the WooCommerce ecommerce plugin. You can easily customize colors, branding and layout options from within the customizer

This theme is compatible with Wordpress Version 3.4 and above and it supports the new theme customization API (https://codex.wordpress.org/Theme_Customization_API).

All the image graphics and icons bundled with this theme are created by the author and licensed under the GPL.

Supported browsers: Firefox, Opera, Chrome, Safari and IE9+ (Some css3 styles like shadows and rounder corners are not supported by IE9 and below).
